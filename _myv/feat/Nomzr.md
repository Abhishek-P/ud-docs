Nominalizer is a feature that indicates verbal derivations.

### <a name="Ag">`Ag`</a>: Agent

The agent nominalizer, previously indicated by `Derivation=NomAg` or `VerbForm=NomAg` has been migrated here. 

#### Examples

* _<b>чав-ыця</b>&nbsp;_ “ Kill-er”
* _<b>чав-ы</b>&nbsp;_ “ Kill-er”
* _<b>чав-ый</b>&nbsp;_ “ Kill-er”
