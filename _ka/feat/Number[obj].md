`Number[obj]` is an inflectional feature of [verbs](_ka/pos/VERB).

### <a name="Sing">`Sing`</a>: singular number

#### Examples

* _მწერ_ 'you write me', _მხატავ_ 'you draw me' etc.

### <a name="Plur">`Plur`</a>: plural number

#### Examples

* _გვწერ_ 'you write us', _გვხატავ_ 'you draw us' etc.
