The `AdvType` feature typically applies to [adverbs](_ka/pos/ADV).

### <a name="Loc">`Loc`</a>: local adverb

#### Examples

* _იქ_ 'there', _აქ_ 'here' etc.

### <a name="Tim">`Tim`</a>: temporal adverb

#### Examples

* _გუშინ_ 'yesterday', _ხვალ_ 'tomorrow' etc.

### <a name="Man">`Man`</a>: manner adverb

#### Examples

* _ძალიან_ 'very', _ცალ-ცალკე_ 'separately' etc.

### <a name="Deg">`Deg`</a>: quantitative adverb

#### Examples

* _ამდენჯერ_ 'so many times', _იმდენჯერ_ 'so often' etc.

### <a name="Cau">`Cau`</a>: causal adverb

#### Examples

* _ამიტომ_ 'that's why', _იმიტომ_ 'because' etc.

### <a name="Int">`Int`</a>: interrogative adverb

#### Examples

* _როდემდე_ 'for how long', _სადამდე_ etc.

### <a name="Rel">`Rel`</a>: relative adverb

#### Examples

* _სადაც_ 'where' etc.
