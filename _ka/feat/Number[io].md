`Number[io]` is an inflectional feature of [verbs](_ka/pos/VERB).

### <a name="Sing">`Sing`</a>: singular number

#### Examples

* _მიწერს_ 'he/she/it writes smth. to me', _მიხატავს_ 'he/she/it is drawing smth. to me' etc.

### <a name="Plur">`Plur`</a>: plural number

#### Examples

* _გვიწერს_ 'he/she/it writes smth. to us', _გვიხატავს_ 'he/she/it is drawing smth. to us' etc.
