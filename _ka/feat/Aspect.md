Aspect is a feature that specifies duration of the action in time, whether the action has been completed etc. The `Aspect` feature applies to [VERB](_ka/pos/VERB).

### <a name="Imp">`Imp`</a>: progressive, imperfective aspect

The action took / takes / will take some time span and there is no information whether and when it was / will be completed.

#### Examples

* _ბავშვი <b>ასრულებს</b> წერას._ 'A child finishes writing.'

### <a name="Perf">`Perf`</a>: perfective aspect

The action has been / will have been completed.

#### Examples

* _ბავშვმა <b>დაასრულა</b> წერა._ 'A child finished writing.'
