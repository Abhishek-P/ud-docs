The `NumForm` feature applies to [numerals](_ka/pos/NUM). In addition to words, digits, combinations, and Roman numeral forms, Georgian also features 'acrophonic numerals,' where the symbols for the numerals are derived from the initial letters of the number names. These acrophonic numerals were represented in Old and Middle Georgian using the _Asomtavruli_, _Nuskhuri_, and _Mkhedruli_ scripts.

### <a name="Word">`Word`</a>: number expressed as a word

#### Examples

* _ერთი_ “one”, _ორი_ “two”, _სამი_ “three”

### <a name="Digit">`Digit`</a>: number expressed using digits

#### Examples

* _1, 2, 3_

### <a name="Combi">`Combi`</a>: digits combined with a suffix

#### Examples

* _1-ი “1st”, მე-11-ე “11th”_

### <a name="Roman">`Roman`</a>: Roman numeral

#### Examples

* _I, II, III_

### <a name="Georgian">`Georgian`</a>: Georgian numeral

#### Examples

* _ა '1', ბ '2', გ '3' etc._
* _ⴀ '1', ⴁ '2', ⴂ '3' etc._
* _Ⴀ '1', Ⴁ '2', Ⴂ	'3' etc._
