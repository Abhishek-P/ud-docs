The `NameType` feature applies to [proper nouns](_ka/pos/PROPN).


### <a name="Sur">`Sur`</a>: surnames

#### Examples

* _შანიძე_ 'Shanidze', _ჭავჭავაძე_ 'Chavchavadze' etc.

### <a name="Prs">`Prs`</a>: personal names

#### Examples

* _მარია_ 'Maria', _ქეთევან_ 'Ketevan' etc.

### <a name="Geo">`Geo`</a>: geographical names

#### Examples

* _თბილისი_ 'Tbilisi', _მტკვარი_ 'Mtkvari' etc.

### <a name="Oth">`Oth`</a>: other names

#### Examples

* _გაერო_ 'UN' (United Nations) etc.
