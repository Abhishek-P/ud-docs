`Number[subj]` is an inflectional feature of [verbs](_ka/pos/VERB).

### <a name="Sing">`Sing`</a>: singular number

#### Examples

* _ვწერ_ 'I write', _ვხატავ_ 'I draw' etc.

### <a name="Plur">`Plur`</a>: plural number

#### Examples

* _ვწერთ_ 'we write', _ვხატავთ_ 'we draw' etc.
