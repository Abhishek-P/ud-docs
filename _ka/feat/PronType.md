The `PronType` feature typically applies to [pronouns](_ka/pos/PRON).

### <a name="Prs">`Prs`</a>: personal pronoun

See also the [Poss](_ka/feat/Poss) feature that distinguishes normal personal pronouns from possessives. possessive pronouns ('Poss=Yes')

#### Examples

* _მე_ 'I', _შენ_ 'you' etc.

### <a name="Dem">`Dem`</a>: demonstrative pronoun

#### Examples

* _ეს_ 'this', _ის_ 'that' etc.

### <a name="Ind">`Ind`</a>: indefinite pronoun

#### Examples

* _ვიღაც_ 'someone', _რაღაც_ 'something' etc.

### <a name="Int">`Int`</a>: interrogative pronoun

#### Examples

* _როგორი_ 'a what sort', _რომელი_ 'which' etc.

### <a name="Tot">`Tot`</a>: determinal pronoun

#### Examples

* _ყოველი_ 'each', _ყველა_ 'every' etc.

### <a name="Neg">`Neg`</a>: negative pronoun

#### Examples

* _არავინ_ 'nobody', _ვერავინ_ 'no-one' etc.

### <a name="Rel">`Rel`</a>: relative pronoun

#### Examples

* _ვინც_ 'who', _რაც_ 'what' etc.

### <a name="Rcp">`Rcp`</a>: reciprocal pronoun

#### Examples

* _ერთმანეთი_ 'each other', _ერთურთი_ 'each other' etc.
