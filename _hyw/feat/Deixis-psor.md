Western Armenian uses `Deixis[psor]` as layered feature for [NOUN]() to encodes the position of an entity relative to either the speaker or the hearer.

### <a name="Prox">`Prox`</a>: proximal

The entity is close to the reference point (e.g., to the speaker).

#### Examples

* _այս <b>վերջերս</b>;_ “...”,
* _դերասաններ են <b>աշխարհիս վրայ</b>;_ “...”,
* _<b>մարդուս</b> աչքերը կը շլանան;_ “...”

### <a name="Med">`Med`</a>: medial

The entity is neither close nor far away from the reference point (e.g., from the speaker).

#### Examples

* _ <b></b>;_ “...”
* _ <b></b>;_ “...”
