`Int` is a 

### <a name="Yes">`Yes`</a>: Yes


Questions in Guajajara follow narow focus, so any constituent can be questioned by attaching the `Int` clitic =_te_ to it

#### Examples

* _<b>Amo=te</b>&nbsp;_ “who?” (person=Q)
* _<b>Maʔe=te</b>&nbsp;_ “What?” (object=Q)
