Boolean feature. Yes is given to non-standalone adverbials ([ADV]()) which are either
loan prefixes (e.g. מולטי ‘multi’, אנטי ‘anti’ etc.) or Hebrew adverbials with the same
function (e.g. בלתי, אי ‘non’).

In other languages, see also [Hyph]() and [Compound]().

### <a name="Yes">`Yes`</a>: it is a non-standalone adverbial/prefix

#### Examples

* <b>מולטי</b> _multi_ “multi”
