This feature indicates, for the head of a [fixed expression](https://universaldependencies.org/en/dep/fixed.html), 
the effective UPOS of the full expression when it differs from the UPOS of the word itself.
Occasionally, `ExtPos` is used even if `ExtPos` = `upos`.

`ExtPos` is used in the [SUD](https://surfacesyntacticud.github.io/) framework.
It is kept in FEATS in the UD conversion.

### <a name="ADJ">`ADJ`</a>: adjective-like expression

#### Examples

* _<b>hors pair</b>_ (_hors_ = `ADP`) "unequalled"

### <a name="ADP">`ADP`</a>: adposition-like expression

#### Examples

* _<b>Il y a</b> un an_ (_Il_ = `PRON`) "One year <b>ago</b>"

### <a name="ADV">`ADV`</a>: adverb-like expression

#### Examples

* _Que demander <b>de plus</b>_ (_de_ = `ADP`) "what more could you ask for"

### <a name="CCONJ">`CCONJ`</a>: coordinating conjunction-like expression

#### Examples

* _<b>ainsi que</b>_ (_ainsi_ = `ADV`) "as well as"

### <a name="DET">`DET`</a>: determiner-like expression

#### Examples

* _le volcan émet <b>de la</b> vapeur_ (_de_ = `ADP`) "volcano emits steam"

### <a name="INTJ">`INTJ`</a>: interjection-like expression

#### Examples

* _<b>Eh bien</b>_ (Eh = `INTJ`) "Well"

### <a name="NOUN">`NOUN`</a>: noun-like expression

#### Examples

* _<b>ras le bol</b>_ (_ras_ = `ADJ`) "fed up"

### <a name="NUM">`NUM`</a>: numeral-like expression

#### Examples

* _<b>k</b>_ (k = `SYM`) "_k_" (mathematical symbol)

### <a name="PRON">`PRON`</a>: pronoun-like expression

#### Examples

* _<b>peu</b> de temps après_ (_peu_ = `ADV`) "shortly after"

### <a name="PROPN">`PROPN`</a>: proper noun-like expression

#### Examples

* _C'est le créateur de <b>The Tenth</b>_ (_The_ = `X`) "He is the creator of The Tenth"

### <a name="SCONJ">`SCONJ`</a>: subordinator-like expression

#### Examples

* _<b>bien que</b>_ (bien = `ADV`) "although"

### <a name="VERB">`VERB`</a>: subordinator-like expression

#### Examples

* _(<b>cf.</b> Carte Zeno)_ (cf. = `X`) "(see Zeno card)"
