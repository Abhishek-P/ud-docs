title: 'ExtPos'
shortdef: 'external POS'

This feature indicates, for the head of a [fixed expression](https://universaldependencies.org/en/dep/fixed.html),
the effective UPOS of the full expression when it differs from the UPOS of the word itself.
Occasionally, `ExtPos` is used even if `ExtPos` = `upos`.

`ExtPos` is used in the [SUD](https://surfacesyntacticud.github.io/) framework.
It is kept in FEATS in the UD conversion.

### <a name="ADJ">`ADJ`</a>: adjective-like expression

#### Examples

* _dis house na <b>no</b> go area_ (_no_ = `AUX`) "This house is a no-go area"

### <a name="ADP">`ADP`</a>: adposition-like expression

#### Examples

* _<b>close</b> to her_ (_close_ = `ADJ`) "Close to her"
* _<b>for</b> inside barrack_ (_for_ = `ADP`) "Inside the barracks"

### <a name="ADV">`ADV`</a>: adverb-like expression

#### Examples

*  _I no dey interested <b>at</b> all_ (_at_ = `ADP`) "I'm not interested at all"
* _<b>of</b> course_ (_at_ = `ADP`) "Of course"
* _<b>at</b> least five percent_ (_at_ = `ADP`) "At least five percent"

### <a name="CCONJ">`CCONJ`</a>: coordinating conjunction-like expression

#### Examples

* _<b>and</b> in fact sey_ (_and_ = `CCONJ`) "And in fact"

### <a name="INTJ">`INTJ`</a>: interjection-like expression

#### Examples

* _<b>na</b> wa for dat woman_ (_na_ = `PART`) "Too bad for that woman"

### <a name="NOUN">`NOUN`</a>: noun-like expression

#### Examples

* _use <b>by</b> force_ (_by_ = `ADP`) "Use by force"

### <a name="PART">`PART`</a>: part-like expression

#### Examples

* _<b>na</b> im be sey_ (_na_ = `PART`) "That's to say"

### <a name="PROPN">`PROPN`</a>: proper noun-like expression

#### Examples

* _<b>Port</b> Harcourt_ (_Port_ = `NOUN`) "Port Harcourt"
* _<b>Cool</b> FM_ (_Cool_ = `ADJ`) "Cool FM"
* _<b>Ben</b> Ten_ (_Ben_ = `PROPN`) "Ben Ten"

### <a name="SCONJ">`SCONJ`</a>: subordinator-like expression

#### Examples

* _<b>na</b> im_ (_na_ = `AUX`) "That's it"

### <a name="VERB">`VERB`</a>: subordinator-like expression

#### Examples

* _e <b>plenty</b>_ (_plenty_ = `ADJ`) "It's plenty"
