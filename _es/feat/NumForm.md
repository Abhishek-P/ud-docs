`NumForm` is a lexical feature of [numerals](NUM) that marks whether the number is expressed by digits or letters.


### <a name="Word">`Word`</a>: number expressed in words

#### Examples

* _uno_ "one"
* _diez_ "ten"
* _cien_ "one hundred"


### <a name="Digit">`Digit`</a>: number expressed using digits

The Digit form is used when the numeral is expressed using digits (0-9).

#### Examples

* _1_ "one"
* _10_ "ten"
* _100_ "one hundred"


### <a name="Roman">`Roman`</a>: roman numeral

#### Examples

* _I_ "one"
* _X_ "ten"
* _C_ "one hundred"
